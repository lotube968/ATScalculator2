# **App Name**: ATS calculator

## Core Features:

- Bill Input: Allow users to input the total bill amount.
- Tipper Assignment: Users assign a tipper who's reponsible to add the tip.
- Tip Percentage Calculation: Calculate the tip amount based on a selected percentage or a custom amount.
- Split Calculation: Calculate the amount each person owes, including the tip.
- Fair Split: Using a generative AI tool that uses Large Language Model to determine the right tip to leave, by analyzing various elements (service, quality, type of restaurant).
- User Interface: Enable users to specify a user id, create a group, and associate their payments in the bill amongst themselves

## Style Guidelines:

- Primary color: Vibrant purple (#9C27B0), aligning with modern aesthetics and providing a rich feel.
- Background color: Soft, light pink (#F8BBD0), desaturated to create a gentle, inviting backdrop.
- Accent color: Fuchsia (#D81B60), adds a lively contrast and highlights interactive elements.
- Font pairing: 'Poppins' (sans-serif) for headlines and 'PT Sans' (sans-serif) for body text, offering a balance of modernity and readability.
- Use minimalist icons with rounded edges for a friendly and approachable feel.
- Clean, card-based layout to separate bill details and calculations clearly.
- Subtle animations for user interactions, such as number transitions and button presses.
